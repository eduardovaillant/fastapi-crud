# Endpoints

| Endpoint    | HTTP Method |	CRUD Method |	Result            |
|-------------|-------------|-------------|-------------------|
| /notes/	    | GET	        | READ	      | get all notes     |
| /notes/:id/ |	GET	        | READ	      | get a single note |
| /notes/	    | POST	      | CREATE	    | add a note        |
| /notes/:id/ |	PUT	        | UPDATE	    | update a note     |
| /notes/:id/ |	DELETE	    | DELETE	    | delete a note     |
